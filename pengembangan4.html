Integrasi Google Spreadsheet

Aplikasi ini dapat mengirimkan data agenda baru secara real-time ke Google Spreadsheet Anda menggunakan fitur Web App Google Apps Script.

Panduan Pengaturan:

  1. Buat sebuah file Google Spreadsheet baru.
  2. Buka menu Ekstensi → Apps Script.
  3. Hapus seluruh kode bawaan, lalu salin-tempel kode Apps Script berikut ini: 
    function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    // Jika sheet masih kosong, tambahkan baris header
    if (sheet.getLastRow() == 0) {
      sheet.appendRow(["ID", "Tanggal", "Jam Mulai", "Jam Selesai", "Mata Pelajaran", "Kelas", "Materi", "Catatan"]);
    }
    
    sheet.appendRow([
      data.id,
      data.tanggal,
      data.jamMulai,
      data.jamSelesai,
      data.mapel,
      data.kelas,
      data.materi,
      data.catatan
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({"result": "success"}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({"result": "error", "message": err.message}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
  4. Klik ikon disket untuk menyimpan proyek script Anda.
  5. Klik tombol Terapkan (Deploy) → Penerapan Baru.
  6. Pilih Jenis: Aplikasi Web (Web App).
  7. Setel opsi "Yang memiliki akses" menjadi "Siapa saja" (Anyone), lalu klik Terapkan.
  8. Salin URL Aplikasi Web yang dihasilkan dan tempelkan ke form input di bawah ini.

Pastikan URL diawali dengan https://script.google.com/
