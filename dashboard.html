Pengaturan Sistem & Manajemen Database

Gunakan alat pengaman ini untuk mengekspor atau mereset seluruh data master secara aman.

Cadangkan Data (Backup)

Ekspor seluruh data akademik saat ini (Siswa, Guru, Absensi, Nilai, Kuis) menjadi file cadangan `.json` ke komputer Anda.

Bersihkan / Reset Data Master

Mengosongkan seluruh database master. Sistem akan secara otomatis mengunduh file backup terlebih dahulu sebelum database di-reset untuk memastikan Anda memiliki salinan cadangan.
